# ci-build-automation-lab

This repository project demonstrates concepts of build automation and CI/CD workflows.

## Overview

- Simulates a firmware or software build using a Bash script
- Simulates a CI pipeline step using Python
- Demonstrates logging, error handling, and build status reporting

## Technologies Used

- Bash scripting
- Python
- Linux command-line tools
- CI/CD concepts (Jenkins-style workflow)

## Files

- `simulate_build.sh` – Simulates a build process (setup, compile, package)
- `ci_step.py` – Simulates a CI pipeline step that runs and evaluates the build

## Purpose

This project was created as a hands-on learning exercise to build
foundational understanding of:
- Build workflows
- Automation infrastructure
- CI/CD pipelines
- Debugging and log analysis
